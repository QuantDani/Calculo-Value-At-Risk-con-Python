# Calculo-Value-At-Risk-con-Python
¿Sabes cuál es la pérdida máxima que podría sufrir tu cartera la próxima semana? Este repositorio busca tratar la gestión de riesgo financiero calculando el Value at Risk (VaR) con Python. En el código muestro paso a paso  los tres métodos fundamentales para calcularlo: Histórico, Paramétrico y la avanzada Simulación de Montecarlo. Analizo la cartera de las '7 Magníficas' aplicando matemáticas rigurosas como la descomposición de Cholesky y valido el modelo con un backtesting profesional para ver si realmente funciona.
